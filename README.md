 Merchant Ops Triage Agent — Groupon Madrid

AI-first intelligent request routing agent built for the Groupon 
Chief of Staff assignment.

## How to run
1. Download `merchant_triage_agent.html`
2. Open it in Chrome or Firefox
3. The agent runs directly in the browser via the Anthropic API

## What it does
- Single request triage with live tool use and multi-step reasoning
- Batch processing of 24 real requests from the assignment dataset
- Routes requests across 5 categories: Auto-Resolve, BPO Standard, 
  In-House Specialist, Manager Escalation, Urgent Escalation
